## The core concept students needed to understand

Using AI as a tool and having control over it, rather than it controlling them. Checking the output manually and through unit testing to ensure quality. Re-prompting and checking output against what is expected. 

## Where students are most likely to struggle

Working with virtual environments for running pytest. Starting from scratch and initializing a repository rather than forking.

## Where AI was helpful vs misleading

AI was helpful in using mermaid.js for the UML format, but was misleading in initially trying to use unittest instead of pytest. 

## One way I would guide a student without giving the answer

I would tell a student to think of this as an actual app you would develop or use. How should these classes interact? What would you test to make sure your app works as intended?
